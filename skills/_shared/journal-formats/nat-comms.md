# Nature Communications formatting facts

Verify current journal guidelines before final submission.

Key working facts:
- Article body is about 5,000 words including Methods.
- Abstract is 150 words, unstructured, no citations.
- Up to 10 display items in the main article.
- References use standard Nature numbered style.
- Data Availability and Code Availability statements are required when applicable.
- Initial submission commonly uses a single PDF with figures embedded.
