# Published Article Patterns

Use when polishing should improve scientific argument, not just English.

Abstract pattern:
1. field-scale problem or opportunity
2. incomplete existing approaches
3. specific intervention or method
4. decisive result with scale, comparison, or constraint
5. why the result changes what is possible
6. scope, application, or boundary

Introduction pattern: scale -> bottleneck -> prior attempts -> missing capability -> present study.

Results pattern: system overview -> validation -> primary performance -> baseline comparison -> mechanism/diagnostic analysis -> scale-up/generalisation.

Discussion pattern: central advance -> credible evidence -> changed workflow/rule/boundary -> remaining constraints -> future work enabled.

Overclaim checks: soften field-wide solution claims, universal applicability, correlation-as-mechanism, unfair comparisons, and future applications stated as achieved outcomes.
