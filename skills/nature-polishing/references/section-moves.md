# Section Moves

Use after main section logic is decided.

Introduction: importance -> known work -> gap/controversy -> study aim -> value/approach.

Methods: design/materials/data -> procedure -> outcomes -> analysis/statistics -> ethics when relevant.

Results: orient to figure/table/experiment -> main observation -> quantitative detail -> pattern -> brief comparison if needed.

Discussion: main finding -> plausible explanation -> earlier work -> limitations -> implications -> future work.

Conclusion: return to aim -> decisive finding -> contribution/significance -> boundary.

Abstract: context -> gap -> approach -> key result -> implication.

Title: searchable, specific, restrained, defensible.
